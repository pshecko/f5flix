## 2 - Resto de páginas

Siguiendo el estilo de la página deberéis crear tres páginas más. Sois
libres de diseñar cada una de ellas siguiendo ??. Todas las páginas
deben contener el header y el footer para poder navegar entre páginas.

### 2.1 Película concreta y/o Serie desarrollada.

Crea **una** página de detalle de una película o serie donde mostrar
toda la información referente a esta, inspírate en páginas que hayas
visitado.

El diseño de esta página es libre. Debes organizar la información como
mejor te parezca.

Recuerda que para acceder a estas páginas será desde el enlace "ver la
serie" o "el título de la pelicula".

### 2.2 Listado de películas o series

Haz una página que permita listar películas y/o series, de otra manera a
cómo se visualiza en la página principal :

En esta página se podrán listar todas las películas y/o todas las
series. Para cada uno se mostrará una imagen, el título, la edad
recomendada, la puntuación, enlace al detalle....

Pon un mínimo de 8-10 pelis/series y sigue el estilo marcado por la
página principal pero haz tu propio diseño.

### 2.3 Contacto/Encuesta

Crea una página con un formulario para poder darse de alta en la web,
añadir un comentario o para responder una encuesta. Deberá tener mínimo
6 tipos diferentes y usaremos el método GET.

Haz el diseño siguiendo el mismo estilo.
